Instrucciones

Decide

Haz una pregunta y después empujar el widget o agitar el celular por conseguir una respuesta de si/no.

Elige

Usa el botón + para añadir opciones. Escriba una nueva opción o usa el botón "Recent" para añadir una opción antigua. Presiona el botón "Elige" Y Re-Resolver eligirá una de las opciones.

Pregunta

Haz una pregunta y después empujar el widget o agitar el celular por conseguir una respuesta mas verboso que puede ser positiva, negativa o ambigua.

Acerca de

Re-Resolver es un intento de preservación para recuperar Resolver – una aplicación de toma de decisiones creado en Islandia que fue una vez altamente calificado – de olvido total causado por actualizaciones de sistemas. El codigo es código abierto y disponible en GitHub.

Aviso

Re-Resolver no tiene ninguna inteligencia de toma de decisiones, se basa en un proceso que es estadísticamente similar a la aleatoriedad. No usas Re-Resolver para tomar decisiones importantes.

Como todos los softwares Re-Resolver contiene errores. Nunca usa tecnología para resolver cosas críticas.
